Avances del Proyecto

Avance 1 — Diseño Inicial

En el primer informe, se diseñó un brazo robótico con un sistema de riego por goteo. Este incluye un sensor de humedad que activa el riego cuando la humedad baja al 50%.

Informe de Proceso — 20/08/2025

Se inició con un prototipo en cartón para visualizar la estructura del brazo. Fue un reto diseñar los ejes de rotación debido al poco conocimiento técnico. Sin embargo, se logró consolidar una idea funcional.

Búsqueda de Componentes — 21/08/2025

Se optó por una punta de rociador de rosca para intercambiarla fácilmente. Se adquirieron dos motores paso a paso para el proyecto.

Diseño 3D — 28/08/2025

El modelado 3D se realizó con asistencia docente y recursos en línea. Fue desafiante por la complejidad de los programas, pero se logró un buen resultado.

Mejoras al Diseño — 17/09/2025

Se rediseñó el brazo para ajustar entradas de motores. Se adaptaron medidas y se consideró la protección contra agua para los componentes electrónicos.

Avance 2 — 01/10/2025

Se realizaron pruebas con servomotores usando Arduino IDE. El servo se conectó con GND, alimentación de 5V y señal PWM desde un pin digital. Se mostró un ejemplo en clase.

Materiales Utilizados

  • 4 servomotores
  • 1 motor paso a paso
  • Filamento TPU flexible
  • Punta de rociador con rosca
  • Placa Arduino UNO R3
  • Cables de conexión
  • Manguera de riego

Software y Conocimientos Necesarios

  • Diseño 3D: Tinkercad, Blender, Autodesk
  • Programación: Python, PHP, MySQL, JavaScript
  • Editor de código: Visual Studio Code

Avance 3 — Próximamente...

En la próxima entrega se presentará el brazo impreso y completamente funcional. Se publicará el diseño gratuitamente para que otros estudiantes lo puedan replicar.
